# 5分钟搞定全平台影视自由

> 花最少的钱，用最简单的方式，看最多的片。

## 你需要什么

- 一部手机/平板/电脑/电视
- 闲鱼 APP
- 10块钱（每月）

## 三步搞定

### 1️⃣ 下载客户端

下载 **网易爆米花**（支持 iOS / Android / Windows / Mac / TV）

- 官网：https://bmh.163.com/
- 用手机号注册登录

### 2️⃣ 闲鱼买资源

打开[闲鱼](https://www.goofish.com/)，搜索以下任意关键词：

- `影视库 emby`
- `网易爆米花 影视`

挑销量高的卖家，**先买1个月试试**（通常 8-10 元）

卖家会发给你：
```
服务器地址：xxx.xxx.xxx
端口：xxxx
用户名：xxxxx
密码：xxxxx
```

### 3️⃣ 配置客户端

1. 打开网易爆米花 → 资源库 →  右上角「➕」加号 → 添加 WebDAV
> 也可能是别的添加方式如emby，具体咨询卖家
2. 填入卖家给的地址、端口、用户名、密码、路径等信息
3. 「媒体库」导航栏 → 点击右上角刷新 → 进行刮削，等待资源同步
4. 完成 ✅

> 💡 刮削完成后，所有设备自动同步，换设备登录同一账号即可。

## 搞定了

现在你拥有了一个随时更新的云端影视库，支持全平台播放。

---

**常见问题**

Q：安全吗？  
A：资源存在卖家服务器，你只是访问。不满意随时换一家。

Q：能看什么？  
A：取决于卖家的库，通常涵盖主流电影、电视剧、动漫、纪录片。买之前可以问卖家要片单。

Q：卡不卡？  
A：取决于卖家服务器和你的网络。好卖家一般不卡。
